# Royals Knowledge Base  
Built with:  
- Vue 3  
- Vite 4  
- Bootstrap 5  
- Axios (latest)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Preview for Production

```sh
npm run preview
```
